Проект создан используя Spring Boot, PostgresSQL DB, и OkHttpClient. Cостоит из двух отдельный модулей:  REST сервера и Http клиента. 
Класс Application в дериктории /project стартует REST сервер. 
Класс HttpClient  в дериктории /project/нttpсlient запускает небольшую программу, где пользователю необходимо ввести в консоль поочередно имя и затем пароль, которые в последствии с помощью POST запроса, в виде JSON будут отправлены на REST сервер, по захардкоженому(чисто для работы программы) url, и сохранятся в баззе данных. 
Функционал для POST(сохранения) данных/пользователя  реализован полностью(правда без всех возможных проверок, но это не основная задача). Функционал для PUT, DELETE, GET реализован процентов на 80.
